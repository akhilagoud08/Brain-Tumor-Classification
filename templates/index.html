Brain Tumor Classification

Upload a brain MRI image and classify it using the best model selected from SVM, Logistic Regression, KNN, Naive Bayes, Decision Tree, and Random Forest.

Upload MRI Image

Preprocessing

Every MRI is converted to grayscale, resized to 200x200 pixels, and normalized before feature extraction.

Feature Methods

The training script compares raw image pixels, Histogram of Oriented Gradients, and Local Binary Pattern features.

Evaluation

Accuracy, precision, sensitivity, specificity, confusion matrix, ROC curve, and classification report are generated.
